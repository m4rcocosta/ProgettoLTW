First of all, we'll start with the introduction of Linux.
Linux is, according to Wikipedia, an Unix-like operative systems' family, published under possible various distributions built around the Linux kernel. We want to illustrate the little differences among the most diffused Linux workstation distributions.
Thanks to the Linux kernel portability, a wide range of operative systems has been developed, for a wide range of devices:

  • personal computer;
  • mobile phones;
  • mainframe;
  • supercomputer.

Linux is really used, mostly as server operative system, in production environments or in embedded devices (PVR, telefoni, ecc.), and it has a sufficient diffusion in desktop environment (about 3% of PCs). Aside it's possible to select the chosen distribution, with its own page containing details and advices about use and installation.
First of all, however, it's important to understand why we advice to install a Linux distribution.
Linux, in its different distributions, is a free software (opensource): this means that everyone can do copies of the software, analyze the source code and modify it.
On the other hand, Linux has a high customizability: you can choose the graphic environment and every graphic environment is developed for a scope.

Moreover, Linux is the only operative system designed and structured in a ""safety" way and that allows a virus immunity near to 100%.
Finally, Linux is really simple: it has a graphic interface very similar to every other operative sistem. In fact, folders, copy and paste, browser are the same or very similar to the programs used on Mac or Windows.
