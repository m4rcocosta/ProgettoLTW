UBUNTU STUDIO


Ubuntu Studio is an official distribution of Ubuntu aimed at amateur and professional multimedia production. Ubuntu Studio uses a different theme from Human, the usual theme of Ubuntu. Ubuntu Studio is designed to provide the user with a set of software for multimedia production, such as Blender for the processing of three-dimensional graphics, Kino and Stopmotion for film postproduction, and other audio processing programs such as Ardor.


Special features


Ubuntu Studio uses a real-time kernel to reduce application latency, during installation allows you to choose whether to install all packages or just some based on the actual needs of the user. The software is divided into: Audio, Video and Graphics. It is also equipped with a personalized audio-visual menu with convenient submenus organized by type, very useful for managing the numerous programs made available in an orderly manner. You can update the operating system online from a standard Ubuntu distribution to Ubuntu Studio by installing the relevant packages.
